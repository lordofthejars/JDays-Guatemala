Qué es Web Profile

Web Profile

  • Introducido en Java EE 6
  • Mitad de especificaciones
  • Libre de sistemas heredados
  • Pero sin algunas especificaciones útiles
Notes
  • "Java EE muy grande, en Java EE 6, la especificación fue partida"
  • "24+ → 12+ Web Profile las mínimas especificaciones para una app"
  • "Sin CMP, Corba, Jax-rpc (Full Profile)"
  • "Pero también sin Jax-rs, jax-ws, jms"
Notes
  • "3 TomEEs (Web Profile + Java mail), JaxRS (merge), TomEE+ (muy por encima de WebProfile pero no Full)"

pom.xml

 <plugin>
    <groupId>org.apache.openejb.maven</groupId>
    <artifactId>tomee-maven-plugin</artifactId>
    <version>1.0.0</version>
    <configuration>
      <tomeeVersion>1.5.0</tomeeVersion>
      <tomeeClassifier>plus</tomeeClassifier>
      <debug>false</debug>
      <debugPort>5005</debugPort>
      <args>-Dfoo=bar</args>
      <config>${project.basedir}/src/test/tomee/conf</config>
      <libs>
        <lib>mysql:mysql-connector-java:5.1.20</lib>
      </libs>
      <webapps>
         <webapp>org.superbiz:api:1.1</webapp>

movieDatabase.JdbcUrl = jdbc:mysql://localhost:3306/moviefun
movieDatabase.UserName = username
movieDatabase.Password = password
movieDatabase.JtaManaged = true

myTransactionManager = new://TransactionManager
myTransactionManager.DefaultTransactionTimeOut = 10 minutes
